About Digital Forensics Project

WCM researchers have an increasing need to analyze digital images to detect prior manipulations.

Cornell Tech students are trying to turn this into a more automated product that could be incrementally improved and commercialized for many industries.

As the problem of image fraud in research papers has become increasingly concerning to the scientific community, we propose to build a website for automatic image manipulation detection. In this work, we conduct user studies to understand desired features that the website should support and research on technical approaches to build an image processing website with an efficient data storage system. So far, we have implemented most of these functionalities and deployed our website. We will continue to work on the website and improve the manipulation detection algorithm next semester.
